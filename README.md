BST Visualizer - 二元搜尋樹互動教學系統
這是一個專為演算法教學設計的 二元搜尋樹 (Binary Search Tree) 互動視覺化工具。使用者可以透過直觀的動畫，理解 BST 的基本操作、遍歷過程以及資料結構的動態變化。

🌟 核心特色
全紀錄快照播放模式：系統在操作開始前會預錄所有執行步驟，使用者可以自由使用「上一步」、「下一步」在時間軸中穿梭，而不會遺失動畫狀態。

自動與手動雙模式：支援「自動播放」功能，並可透過滑桿即時調整動畫速度；也可切換為「手動模式」，逐一確認演算法的每個邏輯步驟。

防衝突保護機制：內建自動重置與計時器管理，當在動畫執行中切換其他功能時，系統會自動中斷舊任務並清理狀態，防止邏輯衝突。

智慧佈局渲染：採用動態座標計算，確保樹狀結構在任何深度都能整齊、對稱地呈現於畫布上。

外部檔案支援：支援讀取 .txt 檔案（以空白或換行分隔數值），快速建立平衡的二元搜尋樹。

🛠 功能說明
1. 基礎操作
插入 (Insert)：搜尋路徑會以黃色高亮顯示，新節點建立時會特別標註。

刪除 (Delete)：視覺化呈現搜尋目標及尋找「後繼者 (Successor)」取代節點的過程。

檔案讀入：自動將數值排序並建立一個高度平衡的初始樹。

2. 遍歷動畫 (Traversal)
支援三種經典遍歷方式，並有獨立的 HTML 容器顯示目前生成的序列：

前序遍歷 (Preorder)

中序遍歷 (Inorder)

後序遍歷 (Postorder)

節點狀態顏色說明：紅色（當前拜訪）、綠色（已紀錄完成）、白色（未拜訪）。

3. 動態控制
進度切換：上一步、下一步、從頭開始。

速度調節：100ms - 1000ms 的延遲控制。

📂 檔案結構
index.html：定義介面結構與控制面板。

style.css：處理畫布響應式佈局與序列文字自動換行。

script.js：核心邏輯，包含繪圖引擎、BST 演算法與快照管理。

🚀 快速上手
下載本專案所有檔案至同一目錄。

使用瀏覽器開啟 index.html。

在輸入框輸入數值按「插入」，或點選「選擇檔案」匯入測試資料。

點擊遍歷按鈕觀察演算法執行細節。

💡 開發備註 (Technical Insights)
在本專案的開發過程中，我們克服了以下挑戰：

非同步同步化：使用 setTimeout 遞迴取代 Promise 阻塞，解決了 UI 凍結問題。

記憶體管理：透過深度複製 (Deep Clone) 樹節點，實作了輕量級的狀態回溯系統。

文字溢出：將動態序列從 Canvas 抽離至 CSS 選取的 HTML 容器，解決了長序列文字超出螢幕邊界的 Bug。
