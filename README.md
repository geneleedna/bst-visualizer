# BST Visualizer - 二元搜尋樹教學演示系統

## 一、 系統功能簡介
* 動態插入與刪除：即時呈現節點增減變化。
* 三大遍歷模式：支援 Preorder, Inorder, Postorder 動畫。
* 步驟回溯系統：具備「上一步」與「下一步」精確控制。
* 播放控制：支援自動播放與 100ms - 1000ms 速度調節。
* 狀態顯示：同步顯示動作描述及遍歷序列。

## 二、 執行方式說明
1. 本地執行：使用瀏覽器直接開啟 index.html。
2. 線上執行：已部署於 GitHub Pages。
   * 網址：https://geneleedna.github.io/bst-visualizer/

## 三、 輸入資料格式說明
1. 手動輸入：在輸入框填入整數，點擊插入或刪除。
2. 檔案載入：點擊「選擇檔案」讀取 .txt 檔案。
   * 格式規範：數值間以「空白」或「換行」分隔。

## 四、 操作流程說明
1. 建立初始樹：透過輸入框或讀取測試 .txt 建立結構。
2. 執行演算法：點擊遍歷按鈕觀察黃/紅/綠節點顏色變化。
3. 觀察與回溯：取消勾選「自動播放」，利用「上一步/下一步」觀察。
4. 重新開始：點擊「從頭開始」回到動畫起始點。
